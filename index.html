Introduction to CSS

CSS (Cascading Style Sheets) is the code that styles web content. CSS basics walks through what you need to get started. CSS is the language we use to style an HTML document, it describes how HTML elements should be displayed.

What is CSS?

Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is what is used to selectively style HTML elements. This is declarative language that controls how webpages look in the browser. The broswer applies CSS style declarations to selected elements to display them properly. A style declaration contains the properties and their values, which determine how a webpage looks.

Anatomy of a CSS ruleset

The whole structure is called a ruleset. The individual parts includes:

  1. Selector: This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled. To style a different element, change the selector.
  2. Declaration: This is a single rule like color: red;. It specifies which element's properties you want to style.
  3. Properties: These are ways in which you can style an HTML element. In CSS, you choose which properties you want to affect in the rule.
  4. Property value

Types of selectors

There are many different types of selectors. Here are some of the more common types of selectors:

  • Element (Tag / Type selector)
  • ID selector
  • Class selector
  • Attribute selector
  • Pseudo-class selector

Summary

Note the important parts of the syntax:

  1. Apart from the selector, each ruleset must be wrapped in curly braces. ({})
  2. Within each declaration, you must use a colon (:) to separate the property from its value or values.
  3. Within each ruleset, you must use a semicolon (;) to separate each declaration from the next one.
